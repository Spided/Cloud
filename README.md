随着云存储技术的普及，高校学生群体对云存储服务的需求日益增长。然而，现
有主流云存储产品存在软件体积臃肿、传输速度限制严重、缺乏面向学生的协作功能
等问题，难以满足学生用户的实际需求。本项目设计并实现了面向高校学生群体的轻
量化协作式云存储客户端软件——轻协云储。
系统采用前后端分离架构，后端基于 ASP.NET Core 框架构建 RESTful API 服务，
前端使用 Qt 框架开发跨平台桌面客户端，数据存储选用轻量级的 SQLite 数据库。在
功能设计上，系统实现了用户管理、文件管理和团队协作三大核心模块。技术创新方
面，通过哈希值比对实现文件秒传功能，采用分块传输和 HTTP Range 协议支持断点
续传，利用闭包表数据结构优化文件树存储，使用队列遍历算法解决深层目录删除问
题。
